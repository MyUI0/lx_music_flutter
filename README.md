# lx_music_flutter

落雪播放器Flutter版


### 主要功能

- 各大平台歌单、排行榜
- 搜索功能
- 播放功能

主要功能已基本实现，但由于种种原因，lx-music-mobile的音源接口已经失效，目前PC版使用六音的音源接口。想尝试将其转成dart，接入到本项目中。有感兴趣的朋友可以一起试试
